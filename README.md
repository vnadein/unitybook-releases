# UnityBook: IDE

Прототип low-code платформы для быстрой разработки учетных и бизнес-приложений.

## Установка

1.  Клонируйте репозиторий
    ```sh
    git clone https://github.com/vnadein/unitybook.git
    ```
2.  Перейти в папку проекта
    ```sh
    cd unitybook
    ```
3.  Установите пакеты зависимости
    ```sh
    pnpm install
    ```
4.  Запустите сервер разработки
    ```sh
    pnpm run dev
    ```

## Описание Системы

**UnityBook** — это гибкая платформа, позволяющая декларативно описывать структуру данных и бизнес-логику приложений. Вся конфигурация системы хранится в виде метаданных (JSON), что позволяет быстро создавать, модифицировать и расширять функциональность без написания большого количества кода.

Основная архитектура строится на следующих понятиях:

-   **Справочники**: для хранения нормативно-справочной информации (клиенты, товары, склады).
-   **Документы**: для регистрации событий и хозяйственных операций (продажи, закупки, перемещения).
-   **Регистры**: для накопления и агрегации учетных данных, которые изменяются документами.

Система предоставляет широкие возможности для кастомизации через встроенные механизмы, такие как вычисляемые поля, печатные формы и серверные модули объектов.
