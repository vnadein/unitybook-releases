# Реквизит "Дата": Полное руководство по свойствам

Этот документ — исчерпывающее руководство по всем свойствам реквизита с типом "Дата". Информация основана на анализе исходного кода проекта и отражает реальное поведение системы.

Реквизит типа "Дата" предназначен для выбора и хранения даты или даты и времени. В интерфейсе он выглядит как поле, при нажатии на которое появляется календарь.

---


#### `Вариант` (`dateVariant`)
*   **Что делает:** Определяет, что именно будет храниться в поле: только дата или дата вместе со временем.
*   **Как работает:** Это свойство может иметь два значения:
    1.  `date` (используется по умолчанию): В поле можно будет выбрать только дату (день, месяц, год).
    2.  `datetime-local`: В поле можно будет выбрать дату и точное время (часы, минуты).
*   **Пример:** Для поля "Дата рождения" подойдет вариант `date`. Для поля "Время начала события" — `datetime-local`.

#### `По умолчанию` (`defaultValue`)
*   **Что делает:** Задает значение, которое будет автоматически подставлено в поле при создании **нового** объекта.
*   **Как работает:** У этого свойства есть особое поведение для дат. Если в качестве значения по умолчанию указать слово `now`, то при создании нового объекта в поле автоматически подставится **текущая системная дата и время**.
*   **Пример:** В документе "Продажа" для поля "Дата документа" можно установить значение `now`, чтобы оно всегда по умолчанию было заполнено текущей датой.

#### `Обязательный` (`required`)
*   **Что делает:** Делает поле обязательным для заполнения.
*   **Как работает:** К названию поля добавляется красная звездочка (`*`), а пустое поле подсвечивается красной рамкой. Система не позволит сохранить объект, пока дата не будет выбрана.

#### `Только чтение` (`readOnly`)
*   **Что делает:** Запрещает пользователю изменять дату.
*   **Как работает:** Поле становится неактивным (серым), и в него нельзя ввести или выбрать другую дату.

#### `Скрыть` (`hidden`)
*   **Что делает:** Полностью скрывает поле из интерфейса формы.
*   **Как работает:** Поле не отображается, но его значение существует и может использоваться в других частях системы (например, для вычислений или в отчетах).

#### `Автозаполнение` (`autofillFrom`)
*   **Что делает:** Автоматически заполняет поле даты на основе значения из другого объекта по ссылке.
*   **Как работает:** Если в форме есть поле-ссылка (например, "Связанный документ"), то для поля "Срок оплаты" можно настроить автозаполнение из реквизита "Дата" этого связанного документа.
