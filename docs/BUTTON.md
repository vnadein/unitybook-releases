# Реквизит "Кнопка": Полное руководство по свойствам

Этот документ — исчерпывающее руководство по всем возможностям реквизита "Кнопка" (тип `button`). Информация основана на анализе исходного кода проекта и отражает реальное поведение системы.

Реквизит "Кнопка" — это интерактивный элемент, который не хранит данные, а выполняет определенное действие при нажатии. Это мощный инструмент для добавления на форму кастомной бизнес-логики.

---

### Основные свойства

#### `Функция модуля` (`onClickFunction`)
*   **Что делает:** **Это главное свойство кнопки.** Оно определяет, какое действие произойдет при нажатии.
*   **Как работает:** В это свойство вы вписываете название функции (например, `calculateTotal`). Эта функция должна быть написана в [Модуле Формы](./MODULE.md). Когда пользователь нажимает на кнопку, система находит в модуле функцию с этим именем и выполняет ее. Это позволяет реализовать любую логику: от простых расчетов до сложных проверок и запросов к серверу.

---

### Свойства внешнего вида

#### `Вариант` (`variant`)
*   **Что делает:** Определяет визуальный стиль кнопки.
*   **Как работает:** Вы можете выбрать один из предопределенных стилей:
    *   `default`: Стандартная, основная кнопка (обычно залита основным цветом).
    *   `destructive`: Для опасных действий, таких как "Удалить" (красная).
    *   `ghost`: "Призрачная" кнопка без фона и рамок, виден только текст.    
    *   `success`: Кнопка для подтверждения. 
    *   `warning`: Кнопка оранжевого цвета. 


#### `Размер` (`size`)
*   **Что делает:** Определяет размер кнопки.
*   **Как работает:** Доступные варианты:
    *   `default`: Нормальный размер.
    *   `sm`: Маленький.
    *   `lg`: Большой.
    *   `icon`: Очень маленький, квадратный размер, предназначенный для кнопок, у которых есть только иконка без текста.

#### `Источник` (иконка) (`icon`)
*   **Что делает:** Добавляет иконку слева от текста кнопки.
*   **Как работает:** В это свойство нужно вписать название иконки из библиотеки [Lucide](https://lucide.dev/icons/). Например, если указать `save`, рядом с текстом появится иконка дискеты.

#### `Размер` и `Насыщенность шрифта`
*   **Как работает:** В системе нет отдельных свойств для управления только шрифтом. Его размер и насыщенность автоматически определяются выбранными `Вариантом` и `Размером` кнопки.
*   **Для продвинутых пользователей:** Существует свойство `className`, в которое можно передать любые CSS-классы для тонкой настройки внешнего вида, включая размер, цвет и насыщенность шрифта.

---

### Свойства состояния

#### `Только чтение` (`readOnly`)
*   **Что делает:** Делает кнопку неактивной.
*   **Как работает:** Если это свойство установлено в `true`, кнопка становится серой, и на нее невозможно нажать. Это полезно, если действие кнопки должно быть доступно только при выполнении определенных условий.

#### `Скрытый` (`hidden`)
*   **Что делает:** Полностью скрывает кнопку с формы.
*   **Как работает:** Кнопка не будет видна пользователю. Это позволяет динамически показывать или скрывать кнопки в зависимости от данных на форме или прав пользователя.
